A Telegram bot with the following Command is presented:
/start - welcomes the user
/game - play a guess game
/age - calculates the age
/voice - English text to speech converter
/max - finds the maximum number amongst an array of numbers
/argmax - finds the argument of the maximum number amongst an array of numbers
/qrcode - converts string into a QR-Code
/help - show the aforementioned commands
